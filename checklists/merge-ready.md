# Merge Ready

Definition of Done for merge/submit steps.

## Gate Criteria

- Clean git status: READY / NOT READY
- Pushed to feature branch: READY / NOT READY
- PR created with correct title format `type(scope): summary (bead-id)`: READY / NOT READY
- PR body includes bead ID and summary: READY / NOT READY
- CI green on PR: READY / NOT READY
- Squash-merged via `gh pr merge --squash`: READY / NOT READY
